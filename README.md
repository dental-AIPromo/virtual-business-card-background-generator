配信元ファイルは `docs/` 配下にあります。

## できること

- 背景画像を選んでプレビュー表示
- バッジ画像を最大4枚まで追加
- ブラウザ内だけで文字を合成
- PNG をダウンロード

## 使い方

1. 公開ページを開きます
2. 背景画像を選びます
3. 必要ならバッジ画像を選びます
4. 文字を入力します
5. PNG をダウンロードします

## 動作

- 背景画像とバッジ画像は利用者が各自で用意します
- 入力内容、背景画像、バッジ画像は保存しません
- 入力内容、背景画像、バッジ画像はサーバーへ送信しません
